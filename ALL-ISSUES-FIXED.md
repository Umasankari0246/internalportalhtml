## 🎯 **How to Use New Features:**

### **Add Shapes:**
1. Go to **Templates → Visual Editor**
2. Click any shape button in "SHAPES" section
3. Shape appears on canvas automatically
4. Drag to position, select to edit properties

### **Customize Shapes:**
1. **Select shape** on canvas
2. **Use Properties Panel** to:
   - Change shape type (rectangle → circle, etc.)
   - Adjust colors (background & border)
   - Modify size (width & height)
   - Change border thickness

### **Edit Existing Templates:**
1. **Templates → Builder Templates**
2. **Click "✏️ Edit"** on any template
3. **Modify design** with all elements loaded
4. **Save changes** - Updates existing template

### **Create with Images:**
1. **Add image element** from left sidebar
2. **Select image** → "Choose Image" button
3. **Upload your image** (JPG/PNG/GIF)
4. **Save template** - Image included in saved version
5. **View in list** - Image displays properly
