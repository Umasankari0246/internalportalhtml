## 🚀 **How to Use Enhanced Builder:**

### **1. Access Visual Builder**
1. Go to **Templates** page
2. Click **"Template Builder"** tab
3. See the new **3-panel layout**:
   - Left: Elements library
   - Center: Visual canvas
   - Right: Properties panel

### **2. Create Professional Templates**

#### **Add Elements:**
- **Text:** Click "📝 Text" → Type content → Edit properties
- **Image:** Click "🖼 Image" → Upload your own or use samples
- **Button:** Click "🔘 Button" → Add CTAs with custom styling
- **Divider:** Click "➖ Divider" → Add visual separators
- **Spacer:** Click "⬜ Spacer" → Control spacing

#### **Customize Elements:**
1. **Select element** → Click any element on canvas
2. **Edit properties** → Use right panel to customize:
   - Font size, color, weight (text)
   - Upload images, adjust dimensions (images)
   - Button text, colors (buttons)
   - Background colors (dividers, spacers)
3. **Drag to position** → Move elements anywhere on canvas
4. **Change background** → Use color picker in left panel

#### **Professional Tips:**
- **Consistent colors** → Match your brand
- **Good spacing** → Don't overcrowd canvas
- **Clear CTAs** → Make buttons stand out
- **Quality images** → Use high-resolution photos
- **Mobile friendly** → Keep elements within canvas bounds

### **3. Image Upload Enhancement**
- **Click image element** → Properties panel shows current image
- **"Choose Image" button** → Upload JPG/PNG/GIF files
- **Automatic optimization** → Images resized for email
- **Instant preview** → See uploaded image immediately
- **Professional appearance** - Rounded corners and proper sizing

### **4. Background Customization**
- **Color picker** → Full spectrum of colors
- **Live preview** → Canvas updates instantly
- **Professional presets** → White, light gray, brand colors
- **Email compatibility** → Background exports to HTML

### **5. Save & Export**
- **Live preview** → Click "👁 Preview" to test
- **Save template** → Click "💾 Save Template" 
- **HTML export** → Clean, email-compatible code
- **Campaign ready** → Works with email campaigns
