## ⚙️ Setup Your Gmail SMTP:

### Step 1: Create Gmail App Password
1. Go to: https://myaccount.google.com/
2. Enable **2-Step Verification** (if not enabled)
3. Go to: https://myaccount.google.com/apppasswords
4. Select "Mail" for the app
5. Generate **16-character App Password**
6. **Copy this password** (shown only once)

### Step 2: Configure in SHOWBAY
1. **Login** to SHOWBAY: http://localhost:3000
   - Email: `admin@showbay.com`
   - Password: `showbay2024`

2. **Go to Settings** page

3. **Enter SMTP Settings:**
   - SMTP Host: `smtp.gmail.com`
   - SMTP Port: `587`
   - Sender Email: `your@gmail.com` (your actual Gmail)
   - Sender Name: `Your Name`
   - App Password: paste the 16-character password

4. **Click "Save Settings"**

5. **Click "Test Connection"**
   - Should show: "SMTP connection verified successfully!"

### Step 3: Send Real Emails
1. **Go to Campaigns** → Create new campaign
2. **Select template** and contacts
3. **Click "Send Test Email"** → Enter your email
4. **Check your Gmail** - Email should arrive!
5. **Click "Send Bulk"** → Real emails sent to all contacts
